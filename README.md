# 🎓 毕业季 - 数字媒体技术的浪漫

一个融合计算机专业特色的毕业季纪念网站，展示从机械制造到数字媒体技术的成长历程。

## ✨ 特色功能

### 🎮 五场景设计
1. **新手村入口** - 二进制代码雨 + 单人照 + 互动按钮
2. **流星雨转场** - 齿轮、代码片段、像素块的赛博朋克转场
3. **证书殿堂** - 剑来背景 + 6张证书环形旋转展示
4. **回忆长廊** - 双人照和群像的横向滚动展示
5. **毕业寄语** - 彩色对话框组成"毕业"二字

### 🖱️ 鼠标交互特效
- **像素风格光标** - 自定义的赛博朋克光标
- **悬停效果** - 按钮放大、证书清晰化、照片旋转
- **点击特效** - 像素爆炸效果
- **光标轨迹** - 移动时留下渐隐的粒子轨迹

### 🎨 视觉风格
- 赛博朋克蓝紫色系
- 像素艺术 + 代码元素
- 发光边框和粒子效果
- 证书隐私模糊保护

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击「是」按钮开始体验
3. 观看流星雨转场动画
4. 鼠标悬停查看证书详情
5. 滑动查看回忆长廊
6. 点击任意位置重新开始

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 动画和样式
- **JavaScript** - 交互逻辑
- **Canvas** - 代码雨和流星雨效果

## 📱 响应式设计

支持手机和PC端自适应，移动端会自动调整证书大小和布局。

## 🎵 音效（可选）

如需音效，请在 `audio/` 目录下添加：
- `click.mp3` - 点击音效
- `transition.mp3` - 转场音效

## 📝 个性化定制

### 修改毕业祝福语
编辑 `js/certificates.js` 中的 `blessings` 数组：
```javascript
this.blessings = [
    '前程似锦', '未来可期', '不负韶华', '毕业快乐',
    // 添加更多祝福语...
];
```

### 修改颜色主题
编辑 `css/style.css` 中的变量：
```css
:root {
    --primary-color: #00f0ff;    /* 主色调 */
    --secondary-color: #ff00ff;  /* 辅助色 */
    --accent-color: #ffff00;     /* 强调色 */
}
```
